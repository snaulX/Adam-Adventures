# Adam Adventures
Adam Adventures - 3D RPG игра, сделанная на движке Unity, используя язык программирования Mino.
# О чём
В мир главного героя по имени Адам, вторглись монстры - гоблины и демоны. Ему надо победить их, используя меч, посох света - оружие на дальние расстояния и заклинания. Для того, чтобы пройти игру вам надо пройти несколько уровней. По ходу их прохождения надо собирать ресурсы, которые можно после уровня потратить на улучшение качеств и изучение новых заклинаний.
